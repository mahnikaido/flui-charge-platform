# ⚡ Flui Charge Platform

Uma plataforma criada para facilitar a vida de motoristas de carros elétricos 🚗⚡

O projeto possui:

* 📱 App mobile para os motoristas
* 💻 Plataforma web administrativa
* 🌐 Deploy online
* 📍 Dados simulados
* 🎨 Interface responsiva e moderna

Desenvolvido para a Etapa 2 do **Charge Platform Challenge**.

# 📱 App Mobile

O app foi pensado para ajudar motoristas a encontrarem pontos de recarga de forma rápida e simples.

### O que tem no app?

* 🗺️ Google Maps integrado
* ⚡ Pontos de recarga simulados
* 🔎 Busca com filtros
* 🔌 Informações dos carregadores
* 📍 Rota pelo Google Maps
* 🌙 Dark mode
* 🎮 Perfil gamificado
* 📲 Layout responsivo

---

# 💻 Painel Web

A plataforma web foi criada para a equipe administrar os pontos de recarga.

### Funcionalidades

* 📋 Listagem de estações
* ➕ Cadastro de novos pontos
* ✏️ Edição de informações
* 📊 Dashboard simples
* ⭐ Avaliações dos motoristas
* 📱 Responsividade
