# Deploy do Flui

Este projeto e estatico: HTML, CSS, JavaScript e assets. Nao precisa de backend para a Etapa 2.

## Links principais depois do deploy

- Site: `/index.html`
- App mobile: `/mobile.html`
- Plataforma web/admin: `/admin.html`
- Login colaborador: `/colaborador.html`

## Vercel

1. Entre em https://vercel.com.
2. Crie um novo projeto.
3. Importe o repositorio publico do GitHub.
4. Framework preset: `Other`.
5. Build command: deixe vazio.
6. Output directory: deixe vazio ou use `.`.
7. Deploy.

## GitHub Pages

1. Crie um repositorio publico.
2. Envie todos os arquivos desta pasta para a branch `main`.
3. Abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Escolha branch `main` e pasta `/root`.
6. Salve e aguarde o link publico.

## Observacao para a entrega

Coloque no PDF o link do repositorio publico e os links publicados:

- `/index.html`
- `/mobile.html`
- `/admin.html`
